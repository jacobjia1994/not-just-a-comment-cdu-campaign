# Don’t Let It Pass at CDU

This is a static campaign website for CDU students. It focuses on everyday anti-Indigenous racism, stigma, shame, bystander hesitation, cultural load and First Nations social and emotional wellbeing.

The site is structured as a short public awareness and behaviour-change campaign:

- `index.html`: campaign entry, interactive “what happens next” moment, burden chain, everyday racism cards and “learn without loading others” module
- `act.html`: interactive next-move practice and copyable response lines
- `support.html`: CDU, crisis, reporting and social and emotional wellbeing support pathways
- `evidence.html`: evidence basis, benchmark campaigns and selected references for the campaign rationale

Legacy paths `problem.html`, `moments.html` and `watch.html` redirect to the main campaign page.

The website is based on the Story template by HTML5 UP and keeps the required template attribution in the footer. It does not imitate Aboriginal art or use decorative Indigenous motifs; any use of First Nations artwork should involve permission, payment, cultural review and attribution.
