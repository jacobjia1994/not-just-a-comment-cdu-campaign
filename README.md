# Not Just a Comment

This is a static student campaign concept for SWK525 Assessment Item 3. It is designed as a public education and awareness resource for CDU students, focusing on everyday anti-Indigenous racism, stigma, shame and barriers to social and emotional wellbeing.

The site is structured as a small multi-page campaign microsite:

- `index.html`: campaign landing page
- `moments.html`: five everyday racism scenarios
- `act.html`: bystander action toolkit
- `watch.html`: embedded official campaign videos
- `support.html`: support and reporting pathways
- `evidence.html`: evidence basis, visual credits and cultural safety note

The website is based on the Story template by HTML5 UP and keeps the required template attribution in the footer. The public student version does not imitate Aboriginal art or use decorative Indigenous motifs; any real implementation should commission or license artwork from First Nations artists.
